# README usage examples

Store the real usage images referenced by the English and Chinese README
files in this directory.

Current filenames:

- `cli-startup.png`: the interactive CLI startup and configuration flow.
- `market-analyst.png`: the market analyst report panel.
- `social-sentiment-analyst.png`: the social sentiment analyst report panel.
- `news-analyst.png`: the news analyst report panel.
- `research-team-decision.png`: the research team debate and decision panel.
- `research-manager-summary.png`: the research manager summary panel.
- `trading-team-plan.png`: the trading team plan panel.
- `portfolio-manager-decision.png`: the portfolio manager decision panel.

After these files are present, the examples will render automatically on
GitHub.
